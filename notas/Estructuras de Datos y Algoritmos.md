# Estructuras de Datos y Algoritmos

Una estructura de datos se le llama a la ubicacion donde se pueden guardar y organizar datos segun ciertos parametros. Donde el algoritmo es la coleccion de pasos para resolver un problema en particular. Aprender Estructuras de datos y algoritmos nos permitira escribir programas optimizados y eficientes.

Todas las estructuras de cubriran con ejemplos hecho en Python, pero estas estructuras aplican la misma logica para otros lenguajes como C++, Java, Javascript, etc.
